Portofolio

Project Saya

Ini adalah beberapa karya yang pernah saya buat. Semoga bisa memberi gambaran tentang skill dan minat saya.

Aplikasi Pengelola Data Mahasiswa

Aplikasi ini merupakan tugas UAS yang dibangun menggunakan PHP Native dengan fitur CRUD sederhana. Sistem ini digunakan untuk mengelola data mahasiswa seperti biodata, program studi, serta pengolahan data dasar lainnya.

Aplikasi Absensi & Gaji Karyawan

Aplikasi ini dirancang untuk membantu perusahaan dalam mengelola absensi karyawan sekaligus perhitungan gaji bulanan. Dengan fitur filter data berdasarkan bulan dan tahun, aplikasi ini dapat menampilkan riwayat kehadiran sekaligus memastikan perhitungan gaji lebih akurat dan transparan.

Aplikasi E-Warranty PT Denso Manufacturing Indonesia

Aplikasi ini berfungsi sebagai sistem monitoring e-warranty untuk produk Denso. Sistem ini memungkinkan pelacakan status garansi, riwayat klaim, serta analisis data warranty melalui dashboard interaktif. Dengan aplikasi ini, proses pengelolaan warranty menjadi lebih terstruktur, cepat, dan akurat.
